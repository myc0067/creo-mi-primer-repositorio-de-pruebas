Git y Github

Trabajo Práctico

Creo mi primer Repositorio de pruebas en Github

Fundamentos

  • Repositorio: Proyecto que está guardado en github.
  • Commit: Versión o cambio.

  • Inicializando nuestro Github: Click aquí para ver los pasos

Crendo un repositorio en remoto

  1. Crar un repositorio nuevo
  2. Ponerle un nombre al repositorio

Ingresar los siguientes comandos:

  • git init
  • git add .
  • git commit -m "nombre de commit"
  • Si es la primera vez: git remote add origin "url de repositorio"
  • git push
